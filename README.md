# Rail to Gainesville

Community advocacy initiative to extend Brightline passenger rail service from Orlando to Gainesville, Florida.

**Website:** [railtogainesville.org](https://railtogainesville.org)

## About

This site collects demand data (zip codes, ride frequency, community connection) to demonstrate ridership potential for an Orlando-Gainesville rail corridor. Aggregated, anonymized data is shared with transit planners, elected officials, and Brightline.

## Setup

Static site hosted on GitHub Pages. To run locally:

```
open index.html
```

## Images

Place images in the `/images` directory and update references in `index.html`.
